# StreetForceApp 
Стек: NodeJs, ExpressJs, PostgresQL, ReactJs (Не учитывая внешние библиотеки)
Реализовать в будущем: 
                      1)Адаптация личного кабинета
                      2)Подключить API для оплаты товаров
                      3)Исправить все мелкие недочеты проекта
